# DATAVIS COM PYTHON - Utilizando pandas, numpy e matplotlib para visualização de dados

**OBJETIVO**  
O presente projeto foi desenvolvido com intuito de praticar funcionalidades de algumas das mais conhecidas bibliotecas de manipulação e análise de dados de Python (pandas, numpy e matplotlib), aprimorar o manejo com JupyterLab, ambiente em que o projeto foi conduzido e, principalmente, trabalhar com um problema real do cotidiano.
