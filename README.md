# DATAVIS COM PYTHON - Utilizando pandas, numpy e matplotlib para visualização de dados

**OBJETIVO**  
O presente projeto foi desenvolvido com intuito de praticar funcionalidades de algumas das mais conhecidas bibliotecas de manipulação e análise de dados de Python (pandas, numpy e matplotlib), aprimorar o manejo com JupyterLab, ambiente em que o projeto foi conduzido e, principalmente, trabalhar com um problema real do cotidiano.  
 ___  

**METODOLOGIA**  
Os dados utilizados para a análise foram gerados por meio de uma inteligência artificial, afim de se obter aleatoriedade total da base recebida. Algumas colunas do arquivo foram previamente especificadas enquanto que outras a IA adicionou automaticamente como variáveis complementares.
Após a geração, realizei o upload do dataset para o meu repositório no GitHub e este foi importado diretamente para o JupyterLab utilizando o link raw, o que permitiu seu carregamento via Pandas.
A garantia da qualidade dos dados foi realizada utilizando Numpy, como a verificação de valores ausentes. Por fim, toda sua análise foi conduzida no ambiente do JupyterLab.
